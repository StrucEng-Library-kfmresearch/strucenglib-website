Extension points

COMPAS has an extensible architecture based on plugins that allows to customize and extend the functionality of the core framework. Check out the developer guide for additional details.

The following extension points are currently defined:

Category: booleans

  • boolean_union_mesh_mesh()

  • boolean_difference_mesh_mesh()

  • boolean_intersection_mesh_mesh()

Category: install

  • installable_rhino_packages()

  • after_rhino_install()

  • after_rhino_uninstall()

Category: intersections

  • intersection_mesh_mesh()

  • intersection_ray_mesh()

Category: quadmesh

  • quadmesh_planarize()

Category: triangulation

  • delaunay_triangulation()

  • constrained_delaunay_triangulation()

  • conforming_delaunay_triangulation()

Category: trimesh

  • trimesh_gaussian_curvature()

  • trimesh_principal_curvature()

  • trimesh_geodistance()

  • trimesh_isolines()

  • trimesh_massmatrix()

  • trimesh_harmonic()

  • trimesh_lscm()

  • trimesh_remesh()

  • trimesh_remesh_constrained()

  • trimesh_slice()
