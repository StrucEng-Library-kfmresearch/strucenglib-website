process_data

compas_fea.utilities.process_data(data, dtype, iptype, nodal, elements, n)[source]

Process the raw data.

Parameters:

  • data (dict) – Unprocessed analysis results data.

  • dtype (str) – ‘nodal’ or ‘element’.

  • iptype (str) – ‘mean’, ‘max’ or ‘min’ of an element’s integration point data.

  • nodal (str) – ‘mean’, ‘max’ or ‘min’ for nodal data conversion.

  • elements (list) – Node numbers for each element.

  • n (int) – Number of nodes.

Returns:

  • array – Data values for each node.

  • array – Data values for each element.
